关于这个博客

你好，欢迎访问我的个人博客！这里是我在互联网上的一个小角落，用来记录一些想法、分享一些日常。

这个博客是用 GitHub Pages 搭建的，虽然很简单，但对我来说意义非凡。这是我第一次亲手搭建属于自己的网站，从注册账号到写代码再到部署上线，每一步都充满了新鲜感。

未来，我会在这里更新更多内容，可能是读书笔记，可能是观影感，也可能是生活里的一些小事。如果你偶然来到这里，希望能让你感到片刻的放松。

近期的小目标

新的阶段，想给自己定一些小目标，不用太宏大，只要能让自己一点点进步就好。

首先是想把专业课学扎实，尤其是网页开发相关的内容，希望能多写多练，做出更多有意思的小项目。然后是想多读几本书，不用追求数量，能认真读完、有所收获就好。最后是想多出去走走，看看不一样的风景，记录下生活里的美好瞬间。

把目标写在这里，也是一种小小的督促。希望下次再看这篇文字的时候，我已经完成了其中的一些。
